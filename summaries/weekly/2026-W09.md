### galaxy-hub
The Galaxy Hub repository completed a migration from Gridsome to Astro as its site generator, moving old Gridsome code to a maintenance branch and updating the CI pipeline accordingly. Several technical fixes were implemented including resolving date display issues on news and events pages, adding funding information to event listings, and enabling automated schema validation. The team also addressed broken external links and synchronized content with their training materials repository.

The Galaxy Project repository received 9 merged pull requests this week, primarily focused on fixing markdown rendering issues in collection steps and job elements, along with dependency updates like bumping minimatch to version 10.2.2 and disallowing gunicorn 25.1.0. The team also improved the Upload Activity interface by enhancing Type and Reference selectors, added XML formatting utilities to galaxy-tool-util, and made workflow test syntax more consistent with Planemo standards.
